2. Ant Design 的 24 格栅格化系统

xs: <576px 响应式栅格
sm：≥576px 响应式栅格
md: ≥768px 响应式栅格
lg: ≥992px 响应式栅格
xl: ≥1200px 响应式栅格
xxl: ≥1600px 响应式栅格


关于引入Icon 组件：从 4.0 开始，antd 不再内置 Icon 组件，请使用独立的包 @ant-design/icons
新版更改为：
使用图标组件，你需要安装 @ant-design/icons 图标组件包：
npm install --save @ant-design/icons

原：<Icon type="home" />
新：<HomeOutlined />
